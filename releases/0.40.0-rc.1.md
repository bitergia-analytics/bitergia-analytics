## Bitergia Analytics 0.40.0-rc.1 - (2026-06-22)

**New features:**

 * Upgrade to GrimoireLab 1.21.0\
   GrimoireLab 1.21.0 updates Vue plugin version of SortingHat's UI with
   a higher Node.js requirement, support for pulling credentials from
   external secrets managers, better Phabricator/Phorge support, and
   updated dependencies accross all the packages.

The following list describes the changes by component:

## bitergia-analytics-opensearch 0.40.0-rc.1 - (2026-06-22)

**New features:**

 * OpenSearch 3.7 upgrade\
   Update base image for Bitergia Analytics OpenSearch to 3.7. OpenSearch
   3.7 brings significant improvements in Analytics Engine with expanded
   Pipeline Query Language (PPL) support, Tiered Storage (WritableWarm)
   implementation for warm storage tiering, and Pluggable Data Formats
   for multi-format indexing. It also includes performance improvements.

## bitergia-analytics-opensearch-dashboards 0.40.0-rc.1 - (2026-06-22)

**New features:**

 * OpenSearch Dashboards 3.7 upgrade\
   Update base image for Bitergia Analytics OpenSearch Dashboards to
   version 3.7.0. This release includes enhanced features for data
   exploration and visualization.
